Description

Prints information on ASPRS LAS files (versions 1.0, 1.1 and 1.2) using the "libLAS" library. Get more information about this library at http://liblas.org Implementation of this tool builds upon the lasinfo tool of Martin Isenburg. liblas version: 1.2.1
