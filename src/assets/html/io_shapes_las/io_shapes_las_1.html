NameImport LAS Files
ID1
AuthorO. Conrad, V. Wichmann (c) 2009

Description

This tool imports ASPRS LAS files (versions 1.0, 1.1 and 1.2) as Point Clouds using the "libLAS" library. Get more information about this library at http://liblas.org Besides the x,y,z coordinates, the user can decide upon which attributes should be imported from the LAS file. Tool development was supported by http://Laserdata GmbH, Austria and http://alpS GmbH, Austria liblas version: 1.2.1
