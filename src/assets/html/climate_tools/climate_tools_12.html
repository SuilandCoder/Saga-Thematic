Tool

NameSoil Water Balance
ID12
AuthorO.Conrad, L.Landschreiber (c) 2016
Specificationgrid, interactive

Description

This tool calculates the water balance for the selected position on a daily basis. Needed input is monthly data of mean, minimum, and maximum temperature as well as precipitation.

References

  • Paulsen, J. / Körner, C. (2014): A climate-based model to predict potential treeline position around the globe. Alpine Botany, 124:1, 1?2. doi:10.1007/s00035-014-0124-0. online.
